## 개요

- 주제선정 : 편의점 알바 경험들이 있어서 POS기를 다루는 것에 대한 기본적인 것들을
          소개하고 경험할 수 있는 프로그램이 있으면 좋을 것 같다는 생각에 이 프로그램을 작성했습니다.

- 주제소개 : 편의점 알바생들이 POS기계를 사용하는 것을 연습해 볼 수 있는 프로그램으로
            알바를 하면서 가장 기본적인 일인 물건 판매, 재고 조사, 시재 점검 등을 프로그램으로 만들어 보았습니다.
